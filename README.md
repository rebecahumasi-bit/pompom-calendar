# 🐶 Calendário Pompompurin

Um calendário interativo com o tema do Pompompurin, feito com HTML, CSS e JavaScript puro. Além de mostrar o mês, ele tem bloco de notas, ferramentas de desenho e uma paleta de cores para você personalizar cada dia.

## ✨ Funcionalidades

- **Calendário mensal** com o dia atual em destaque e mini-calendários do mês anterior e do próximo.
- **Navegação entre meses** pelos botões de avançar e voltar.
- **Lista diária** na barra lateral para acompanhar os dias.
- **Bloco de notas** com área de texto (até 500 caracteres).
- **Ferramentas de desenho** sobre o calendário e as notas: caneta, marcador e paleta de 6 cores.
